# OWASP Top 10

## Broken Access Control
Unauthorized users gaining access to restricted resources.

## Cryptographic Failures
Weak encryption leading to data exposure.

## Injection
Malicious input interpreted as commands by the system.

## Security Misconfiguration
Improper security settings exposing the application.

## Vulnerable Components
Use of outdated software with known vulnerabilities.

## Identification and Authentication Failures
Weak authentication mechanisms leading to account compromise.
